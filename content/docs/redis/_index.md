Redis 是广为流行的开源“高性能”内存数据结构服务器，PostgreSQL 的好搭子。
Pigsty 中的 Redis 是一套生产就绪的完整解决方案，支持主从复制、哨兵高可用与原生集群模式，集成了监控与日志功能，并提供自动化的安装、配置与运维管理剧本。
