PostgreSQL 集群级别监控面板，包括：

- [PGSQL Cluster](/docs/pgsql/dashboard/cluster/pgsql-cluster)：一个 PGSQL 集群的主仪表板
- [PGRDS Cluster](/docs/pgsql/dashboard/cluster/pgrds-cluster)：PGSQL Cluster 的 RDS 版本，专注于 PostgreSQL 本身的指标
- [PGSQL Activity](/docs/pgsql/dashboard/cluster/pgsql-activity)：关注 PGSQL 集群的会话/负载/QPS/TPS/锁定情况
- [PGSQL Replication](/docs/pgsql/dashboard/cluster/pgsql-replication)：关注 PGSQL 集群复制、插槽和发布/订阅
- [PGSQL Service](/docs/pgsql/dashboard/cluster/pgsql-service)：关注 PGSQL 集群服务、代理、路由和负载均衡
- [PGSQL Databases](/docs/pgsql/dashboard/cluster/pgsql-databases)：关注所有实例的数据库 CRUD、慢查询和表统计信息
- [PGSQL Patroni](/docs/pgsql/dashboard/cluster/pgsql-patroni)：关注集群高可用状态，Patroni 组件状态
- [PGSQL PITR](/docs/pgsql/dashboard/cluster/pgsql-pitr)：关注集群 PITR 过程的上下文，用于辅助时间点恢复
