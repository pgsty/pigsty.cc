PostgreSQL 实例级别监控面板，包括：

- [PGSQL Instance](/docs/pgsql/dashboard/instance/pgsql-instance)：单个 PGSQL 实例的主仪表板
- [PGRDS Instance](/docs/pgsql/dashboard/instance/pgrds-instance)：PGSQL Instance 的 RDS 版本，专注于 PostgreSQL 本身的指标
- [PGCAT Instance](/docs/pgsql/dashboard/instance/pgcat-instance)：直接从数据库目录获取的实例信息
- [PGSQL Persist](/docs/pgsql/dashboard/instance/pgsql-persist)：持久性指标：WAL、XID、检查点、存档、IO
- [PGSQL Proxy](/docs/pgsql/dashboard/instance/pgsql-proxy)：单个 HAProxy 负载均衡器的详细指标
- [PGSQL Pgbouncer](/docs/pgsql/dashboard/instance/pgsql-pgbouncer)：单个 Pgbouncer 连接池实例中的指标总览
- [PGSQL Session](/docs/pgsql/dashboard/instance/pgsql-session)：单个实例中的会话和活动/空闲时间的指标
- [PGSQL Xacts](/docs/pgsql/dashboard/instance/pgsql-xacts)：关于事务、锁、TPS/QPS 相关的指标
- [PGSQL Exporter](/docs/pgsql/dashboard/instance/pgsql-exporter)：Postgres 与 Pgbouncer 监控组件自我监控指标
