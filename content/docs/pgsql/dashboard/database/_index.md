---

PostgreSQL 数据库级别监控面板，包括：

- [PGSQL Database](pgsql-database)：单个 PGSQL 数据库的主仪表板
- [PGCAT Database](pgcat-database)：直接从数据库目录获取的数据库信息
- [PGSQL Tables](pgsql-tables)：单个数据库内的表/索引访问指标
- [PGSQL Table](pgsql-table)：单个表的详细信息（QPS/RT/索引/序列……）
- [PGCAT Table](pgcat-table)：直接从数据库目录获取的单个表的详细信息
- [PGSQL Query](pgsql-query)：单类查询的详细信息（QPS/RT）
- [PGCAT Query](pgcat-query)：直接从数据库目录获取的单类查询的详细信息
- [PGCAT Locks](pgcat-locks)：直接从数据库目录获取的关于活动与锁等待的信息
- [PGCAT Schema](pgcat-schema)：直接从数据库目录获取关于模式的信息
