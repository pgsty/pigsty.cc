`pg_lake_iceberg` еңЁPostgreSQLеҶ…йғЁе®һзҺ°дәҶIcebergзҡ„е…ғж•°жҚ®гҖҒеҝ«з…§гҖҒжё…еҚ•гҖҒеҲҶеҢәи§„иҢғе’Ңзӣ®еҪ•йӣҶжҲҗгҖӮдҫқиө–з»„д»¶ `CREATE TABLE ... USING iceberg` жҡҙйңІдәҶзҶҹжӮүзҡ„ `pg_lake_table` иҜӯжі•пјӣз”ЁжҲ·йҖҡеёёйҖҡиҝҮ `pg_lake` е®үиЈ…иҝҷдёӨдёӘз»„д»¶гҖӮ

### еҲӣе»әе№¶жЈҖжҹҘдёҖдёӘIcebergиЎЁ

```sql
CREATE EXTENSION pg_lake CASCADE;

SET pg_lake_iceberg.default_location_prefix =
    's3://analytics-bucket/warehouse';

CREATE TABLE events (
    event_time timestamptz NOT NULL,
    user_id bigint NOT NULL,
    payload jsonb
) USING iceberg
WITH (partition_by = 'day(event_time), bucket(32, user_id)');

SELECT table_namespace, table_name, metadata_location
FROM iceberg_tables
WHERE table_name = 'events';
```

жЈҖжҹҘIcebergе…ғж•°жҚ®ж–Үд»¶еҸҠе…¶еј•з”Ёзҡ„зҠ¶жҖҒпјҡ

```sql
SELECT lake_iceberg.metadata(metadata_location)
FROM iceberg_tables
WHERE table_name = 'events';

SELECT f.*
FROM iceberg_tables AS t
CROSS JOIN LATERAL lake_iceberg.files(t.metadata_location) AS f
WHERE t.table_name = 'events';
```

### е…ғж•°жҚ®е’Ңзӣ®еҪ•API

- `iceberg_tables`: `pg_catalog` и§ҶеӣҫпјҢз»“еҗҲжң¬ең°з®ЎзҗҶиЎЁе’ҢеӨ–йғЁзӣ®еҪ•жқЎзӣ®гҖӮ
- `iceberg_namespace_properties`: зӣ®еҪ•е‘ҪеҗҚз©әй—ҙеұһжҖ§гҖӮ
- `lake_iceberg.metadata(uri)`: еҺҹе§ӢIcebergе…ғж•°жҚ®JSONгҖӮ
- `lake_iceberg.files(uri)`: жё…еҚ•и·Ҝеҫ„гҖҒеҶ…е®№зұ»еһӢгҖҒж•°жҚ®ж–Үд»¶и·Ҝеҫ„/ж јејҸгҖҒи§„иҢғIDгҖҒи®°еҪ•ж•°е’Ңж–Үд»¶еӨ§е°ҸгҖӮ
- `lake_iceberg.snapshots(uri)`: еәҸеҲ—еҸ·гҖҒеҝ«з…§IDгҖҒж—¶й—ҙжҲіе’Ңжё…еҚ•еҲ—иЎЁи·Ҝеҫ„гҖӮ
- `lake_iceberg.data_file_stats(uri)`: жҜҸдёӘж–Үд»¶зҡ„еәҸеҲ—еҸ·еҸҠдёӢйҷҗ/дёҠйҷҗпјӣжү§иЎҢжқғйҷҗжҺҲдәҲ `lake_read` иҖҢйқһ `PUBLIC`гҖӮ
- `iceberg_catalog`: зүҲжң¬3.4 FDWпјҢз”ЁдәҺе‘ҪеҗҚPostgreSQLгҖҒеҜ№иұЎеӯҳеӮЁжҲ–RESTзӣ®еҪ•й…ҚзҪ®гҖӮ

е®ҡд№үдёҖдёӘз”ЁжҲ·з®ЎзҗҶзҡ„RESTзӣ®еҪ•жңҚеҠЎеҷЁпјҢе№¶е°ҶеҮӯжҚ®дҝқеӯҳеңЁз”ЁжҲ·жҳ е°„дёӯпјҡ

```sql
CREATE SERVER my_polaris TYPE 'rest'
FOREIGN DATA WRAPPER iceberg_catalog
OPTIONS (rest_endpoint 'https://polaris.example.com');

CREATE USER MAPPING FOR app_role SERVER my_polaris
OPTIONS (client_id 'app', client_secret 'secret');

CREATE TABLE catalog_events (id bigint)
USING iceberg
WITH (catalog = 'my_polaris');
```

### зӣ®еҪ•е’ҢеӯҳеӮЁжіЁж„ҸдәӢйЎ№

- з”ЁжҲ·еҲӣе»әзҡ„зӣ®еҪ•жңҚеҠЎеҷЁйңҖиҰҒиҮӘе·ұзҡ„ `USER MAPPING` еҮӯиҜҒпјҢдёҚдјҡеӣһйҖҖеҲ°еҶ…зҪ®зҡ„RESTзӣ®еҪ•еҮӯиҜҒGUCгҖӮ
- еҶ…зҪ®зҡ„ `postgres`гҖҒ`object_store` е’Ң `rest` зӣ®еҪ•жҳ е°„еҲ°дёҚеҸҜеҸҳзҡ„жү©еұ•жӢҘжңүиҖ…жңҚеҠЎеҷЁгҖӮйҖҡиҝҮж–ҮжЎЈдёӯзҡ„GUCиҝӣиЎҢй…ҚзҪ®пјҢиҖҢдёҚжҳҜдҝ®ж”№иҝҷдәӣжңҚеҠЎеҷЁгҖӮ
- еҜ№ `iceberg_tables` зҡ„еӨ–йғЁдҝ®ж”№й»ҳи®Өиў«йҳ»жӯўпјҢеӣ дёәжӣҙж”№е…ғж•°жҚ®еҸҜиғҪдјҡз ҙеқҸpg_lakeзҡ„дәӢеҠЎе’ҢжҹҘиҜўеј•ж“ҺдёҖиҮҙжҖ§гҖӮ
- IcebergеҶҷж“ҚдҪңеә”еҲҶжү№жү§иЎҢгҖӮжҜҸж¬ЎиҜӯеҸҘеҸҜд»Ҙж·»еҠ Parquetж–Үд»¶е’Ңеҝ«з…§пјӣеёёи§„ `VACUUM` дјҡеҺӢзј©е°Ҹж–Үд»¶е№¶ж №жҚ®иЎЁ/GUCзӯ–з•ҘиҝҮжңҹж•°жҚ®гҖӮ
- IcebergеҜ№жҹҗдәӣPostgreSQLеҖјжңүжӣҙзӘ„зҡ„иЎЁзҺ°еҪўејҸгҖӮй»ҳи®Өзҡ„ `out_of_range_values = 'error'` дҝқжҢҒе®Ңж•ҙжҖ§пјӣ`clamp` й»ҳй»ҳж”№еҸҳи¶…еҮәиҢғеӣҙзҡ„ж—¶й—ҙжҲіеҖјпјҢе№¶з”Ё `NULL` жӣҝжҚўдёҖдәӣдёҚж”ҜжҢҒзҡ„еҖјгҖӮ
