### жҹҘиҜўеӨ–йғЁж–Үд»¶

е®үиЈ…е®Ңж•ҙзҡ„е Ҷж ҲпјҢеҢ…жӢ¬жүҖйңҖзҡ„ `pg_lake_engine`гҖҒ`pg_lake_iceberg` е’Ң `btree_gist` дҫқиө–йЎ№пјҡ

```sql
CREATE EXTENSION pg_lake CASCADE;
```

з©әеҲ—еҲ—иЎЁиҰҒжұӮ pg_lake жҺЁж–ӯж–Үд»¶жЁЎејҸпјҡ

```sql
CREATE FOREIGN TABLE external_events ()
SERVER pg_lake
OPTIONS (
    path 's3://analytics-bucket/events/*.parquet',
    filename 'true'
);

SELECT count(*) FROM external_events;
```

йҖҡиҝҮжү©еұ•жҸҗдҫӣзҡ„иЎЁи®ҝй—®ж–№жі•еҲӣе»әеҸҜеҶҷзҡ„ Iceberg иЎЁпјҡ

```sql
CREATE TABLE managed_events (
    event_time timestamptz,
    payload jsonb
) USING iceberg;
```

### ж–Үд»¶е’ҢиЎЁе®һз”Ёе·Ҙе…·зҙўеј•

- `lake_file.list(pattern)`: еҲ—еҮәеҢ№й…Қзҡ„еҜ№иұЎи·Ҝеҫ„гҖҒеӨ§е°ҸгҖҒдҝ®ж”№ж—¶й—ҙе’Ң ETagsгҖӮ
- `lake_file.size(path)` е’Ң `lake_file.exists(path)`: жЈҖжҹҘдёҖдёӘиҝңзЁӢеҜ№иұЎгҖӮ
- `lake_file.preview(url, format, compression)`: иҝ”еӣһжҺЁж–ӯзҡ„еҲ—еҗҚе’Ңзұ»еһӢгҖӮ
- `lake_file.delete(url)`: еҲ йҷӨдёҖдёӘиҝңзЁӢеҜ№иұЎпјӣд»…йҷҗе…Ғи®ёеҲ йҷӨж•°жҚ®зҡ„и§’иүІжү§иЎҢгҖӮ
- `lake_file_cache.add(path, refresh)`, `remove(path)` е’Ң `list()`: з®ЎзҗҶ `lake_read` жҲҗе‘ҳзҡ„жң¬ең°ж–Үд»¶зј“еӯҳгҖӮ
- `lake_iceberg.table_size(regclass)`: и®Ўз®— Iceberg иЎЁеҪ“еүҚж•°жҚ®ж–Үд»¶зҡ„жҖ»еӨ§е°ҸгҖӮ
- еӨ–йғЁжңҚеҠЎеҷЁ `pg_lake` е’Ң `pg_lake_iceberg`: иҜ»еҸ–ж–Үд»¶е’ҢеҸҜеҶҷзҡ„ Iceberg е…ҘеҸЈзӮ№гҖӮ
- и®ҝй—®ж–№жі• `iceberg` е’Ң `pg_lake_iceberg`: з”ЁдәҺе°Ҷ `CREATE TABLE ... USING` иҪ¬жҚўдёәжү©еұ•зҡ„еӨ–йғЁиЎЁиЎЁзӨәеҪўејҸзҡ„еҲ«еҗҚгҖӮ

```sql
SELECT path, file_size
FROM lake_file.list('s3://analytics-bucket/events/**/*.parquet');

SELECT *
FROM lake_file.preview('s3://analytics-bucket/events/sample.parquet');
```

### иҝҗиҗҘжіЁж„ҸдәӢйЎ№

- еҝ…йЎ»йў„еҠ иҪҪ `pg_extension_base` е№¶еңЁжҜҸдёӘеј•з”ЁдҪҚзҪ®иҝҗиЎҢ `pgduck_server`пјҢе№¶еёҰжңүзӣёеә”зҡ„еҮӯжҚ®гҖӮ
- `lake_read`гҖҒ`lake_write` е’Ң `lake_read_write` жҺ§еҲ¶еҜ№еӨ–йғЁжңҚеҠЎеҷЁгҖҒжЁЎејҸе’Ңе®һз”Ёе·Ҙе…·зҡ„и®ҝй—®гҖӮдёәжҜҸдёӘеә”з”ЁзЁӢеәҸжҺҲдәҲжңҖзӘ„зҡ„и§’иүІжқғйҷҗгҖӮ
- еӨ–йғЁиЎЁжҳҜж–Үд»¶зҡ„еј•з”ЁпјҢиҖҢдёҚжҳҜеҜје…Ҙзҡ„еүҜжң¬гҖӮж–Үд»¶жӣҝжҚўгҖҒи·ЁеҢәеҹҹи®ҝй—®е’Ңзј“еӯҳеӨұж•ҲеҸҜд»ҘзӢ¬з«ӢдәҺ PostgreSQL зӣ®еҪ•зҠ¶жҖҒж”№еҸҳе»¶иҝҹжҲ–з»“жһңгҖӮ
- Iceberg жҸ’е…ҘдјҳеҢ–жү№йҮҸжҸ’е…ҘиҖҢйқһеҚ•иЎҢжҸ’е…ҘгҖӮеҜ№дәҺй«ҳйў‘зҺҮйҖҗиЎҢжҸ’е…ҘпјҢдҪҝз”ЁжҡӮеӯҳе ҶиЎЁпјҢе№¶е®ҡжңҹеҲ·ж–°жү№ж¬ЎгҖӮ
- еҶ…йғЁ `lake_table.*` зӣ®еҪ•и·ҹиёӘж–Үд»¶гҖҒеӯ—ж®ө IDгҖҒеҲҶеҢәе’ҢжҒўеӨҚзҠ¶жҖҒгҖӮдёҚиҰҒзӣҙжҺҘдҝ®ж”№е®ғд»¬гҖӮ

ж–ҮжЎЈз»“жқҹ
